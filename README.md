# 🍅 Pomodoro Timer

A modern, interactive Pomodoro productivity timer built with Streamlit.

Stay focused, track your work sessions, and manage tasks effectively.

---

## 🚀 Features

### ⏱️ Timer System

* Work sessions (default 25 mins)
* Short & long breaks
* Automatic session transitions

### 📊 Productivity Tracking

* Completed Pomodoros
* Total focus time
* Daily stats dashboard

### ⚙️ Customization

* Adjustable durations
* Configurable long-break intervals

### 📝 Task Management

* Add and track daily tasks
* Mark tasks as complete
* Delete tasks

### 🔔 Feedback

* Visual notifications
* Session completion alerts
* Progress bar

---

## 🛠️ Tech Stack

* Python
* Streamlit
* pygame (for sound notifications)

---

## 📦 Installation

```bash
git clone https://github.com/yourusername/pomodoro-timer.git
cd pomodoro-timer

pip install -r requirements.txt
```

---

## ▶️ Run the App

```bash
streamlit run pomodoro_timer.py
```

---

## 🧠 How It Works

1. Start a work session
2. Focus until timer ends
3. Take a break
4. Repeat cycle
5. After several sessions → long break

---

## ⚠️ Notes

* Sound notifications require pygame setup
* Timer uses browser refresh loop (Streamlit limitation)
